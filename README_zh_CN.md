# Excel 导入数据库

将 `.csv`、`.xlsx` 或 `.xls` 表格直接批量导入思源笔记数据库。

## 功能

- 在数据库右上角提供 `Excel 导入` 入口。
- 支持 CSV、Excel 97-2003 和 Excel 工作簿。
- 支持选择工作表。
- 根据表头自动匹配数据库字段。
- 可将新列自动创建为文本字段，也可以手动跳过。
- 自动跳过主键为空的行。
- 每 100 行分批写入，适合较大的表格。

## 使用方法

1. 打开一个包含数据库的思源文档。
2. 点击数据库右上角的 `Excel 导入`。
3. 选择表格文件和工作表。
4. 检查字段映射，确认主键已经映射。
5. 点击 `开始导入`。

## 支持的已有字段

插件可以写入主键、文本、数字、单选、多选、复选框、URL、邮箱和电话字段。复杂字段请在映射时跳过。
